Steps to execute the assignment :

1. Import the project org.iiitb.MT2011010.tripPlannerProject in Eclipse.
2. copy the "tripPlanner.sql" into the bin of mysql of program file.
3. run the command in cmd : 
	- go to the location of bin in cmd
	- create database in mysql : tripDetailsData
	- run the command : mysql -u root -p tripDetailsData < tripPlanner.sql
4. Go to RunTimeSettings of mysql and give the password of Your mysql server.
5. Run the project as eclipse Application.
